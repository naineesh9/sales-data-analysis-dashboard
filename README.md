# Sales Data Analysis & Dashboard

## Overview
This project analyzes retail sales data to identify trends, track key performance indicators, and generate business insights that can support better decision-making.

## Tools Used
- Python (Pandas, Matplotlib)
- Power BI
- Excel

## Project Structure
- `data/sales_data.csv` — retail sales dataset
- `notebooks/eda_analysis.ipynb` — cleaning, EDA, KPI analysis
- `images/dashboard_preview.png` — dashboard preview image

## Key Steps
1. Cleaned and transformed sales data
2. Created KPI metrics such as total sales, total profit, total orders, and profit margin
3. Performed exploratory data analysis on regions, categories, and monthly trends
4. Prepared outputs for a Power BI dashboard

## Business Insights
- Highlights high-performing regions and top-selling sub-categories
- Surfaces low-margin areas that may need pricing or discount strategy changes
- Tracks monthly sales trends to support forecasting and planning

## Suggested Power BI Dashboard
Build a dashboard with:
- KPI cards for sales, profit, orders, and profit margin
- Monthly sales trend line chart
- Regional sales bar chart
- Top sub-category chart
- Filters for region, category, segment, and month

## Resume Bullet
Built an interactive Power BI dashboard analyzing sales, profit, and regional performance using retail data; performed data cleaning and exploratory data analysis in Python to generate actionable business insights.

## How to Run
1. Open `notebooks/eda_analysis.ipynb`
2. Run the notebook cells
3. Use the cleaned outputs and insights to build a Power BI dashboard
